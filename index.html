Welcome to the API!

How to use this API:

  • POST /signup : Register a new user. The body should be a JSON object containing 'username' and 'password' keys.
  • POST /login : Authenticate an existing user. The payload should be a JSON object with 'username' and 'password' keys. This endpoint returns a JWT token when successful.
  • GET /logout : Invalidate a user's JWT token. This endpoint needs no payload.
  • GET /reservs : Get all reservations.
  • POST /reservs : Create a new reservation. The body should be a JSON object with 'name', 'description', 'date', 'phone', 'email', 'created_at' and 'user_id' keys.
  • PUT /reservs : Update an existing reservation. The body should be a JSON object with 'name', 'description', 'date', 'phone', 'email' and 'created_at' keys. 'created_at' key should specifically correlate to the post you want to update.
  • DELETE /reservs : Delete a reservation. The body should be a JSON object with 'created_at' key, which should specifically correlate to the post you want to delete.
